# mysql_to_markdown

mysql数据库转成markdown格式，只要输入数据库信息，所有表字段生成markdown

## 环境要求
* Web服务：nginx或Apache
* 版本控制：Git
* 依赖包：composer
* PHP: ^5.6

## USED

### 安装
`git clone https://github.com/mikeah2011/mysql_to_markdown.git`

### 初始化
`composer i`
